このサイトについて

HTML・CSS・JavaScript・jQuery を引いて、その場で動かして確かめられる、無料のコード辞書です。

CoDictionary とは

「読んでも分からない」を「見れば分かる」に変えるための辞書です。プログラミング未経験の方でも、調べたその場で動く物を見て確かめられます。各項目は次の内容をそろえています。

  • タグ／プロパティ／命令の意味とルール・使い方
  • そのコードで使う言葉の解説
  • その場で実際に動くプレビュー
  • 自分で書き換えて試せるコード

項目は 基礎・初級・中級・上級・特級・応用 の6つの難易度に分類しています。順番に進む必要はありません。知りたいところから引いてください。

使い方

  • 言語のボタンで HTML / CSS / JS / jQuery を選ぶ
  • 難易度でしぼるか、検索欄にキーワードを入れて項目を探す（検索は4言語すべてを横断します）
  • 項目を選ぶと、説明・使う言葉・コード・プレビューが表示される
  • コード窓を編集すると、すぐにプレビューへ反映される
  • よく引く項目は ♡ でお気に入りに（「♥ いいね」タブに、言語をまたいでまとまります）

プライバシーポリシー

当サイトは、利用者のプライバシーを尊重し、以下の方針でデータを取り扱います。

  • お気に入り（いいね）：お使いのブラウザ内の localStorage にのみ保存され、外部サーバーには送信されません。ブラウザのデータを消すと消去されます。
  • 個人情報の収集：会員登録・ログイン機能はなく、氏名やメールアドレス等を当サイトが直接収集・保存することはありません。
  • お問い合わせ：お問い合わせフォームは SSGform（国産の外部フォーム送信サービス）を利用しています。送信内容は SSGform 上で処理されます（SSGform のプライバシーポリシーが適用されます）。
  • アクセス解析：閲覧状況の把握のため Counter.dev（プライバシー配慮型のアクセス解析）を利用しています。Cookie は使用せず、個人を特定する情報は取得しません。
  • Cookie / ローカル保存：上記の目的以外で利用者を追跡する目的の Cookie は使用しません。

本ポリシーは予告なく変更される場合があります。最終更新：2026-07-22
